# Load rogtemplate fonts

Load the current font in use for rOpenGov, [B612
Mono](https://fonts.google.com/specimen/B612+Mono)

## Usage

``` r
rog_load_font()
```

``` r
rog_load_font()
#> B612 Mono font loaded
#> [1] "B612 Mono"
```
